## 소개

본 프로젝트는 PBV(Purpose Built Vehicle) 차량의 디스플레이에 노출될 HMI(Human-Machine Interface)를 시뮬레이션한 것이다. PBV 후방 모듈 교체 과정(연결 대기 → 근거리 통신 → 제어권 이양 → 자동 제어 → 완료)에서 발생하는 각 단계별 상태 정보를 운전자에게 직관적으로 시각화하여 전달한다. 모듈 교체 전 과정의 진행 상황을 명확하게 표시함으로써 운전자의 상황 오인(誤認)을 방지하고, 안전하고 효율적인 모듈 교체 환경을 제공한다.
